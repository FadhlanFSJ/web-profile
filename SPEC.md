# Developer Profile Website

## Project Overview
- **Type**: Personal developer portfolio/profile website
- **Core Functionality**: Showcase developer skills, projects, GitHub activity, and contact info
- **Target Users**: Recruiters, hiring managers, fellow developers

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Content**: Static data with GitHub API integration

## Visual & Rendering Specification

### Layout
- Single page with smooth scroll navigation
- Hero section with name, title, and short bio
- Skills/tech stack grid
- Featured projects gallery
- GitHub activity/recent contributions
- Contact section

### Color Palette
- Background: Dark theme (#0a0a0a)
- Primary accent: Electric blue (#3b82f6)
- Secondary accent: Cyan (#22d3ee)
- Text: White/gray scale

### Typography
- Headings: JetBrains Mono or similar monospace
- Body: Inter or system sans-serif

## Features

### Sections
1. **Hero**: Name, title "Full Stack Developer", bio, CTA buttons
2. **Skills**: Grid of tech icons/languages (React, Node.js, TypeScript, Python, etc.)
3. **Projects**: 4-6 featured projects with title, description, tech stack tags, links
4. **GitHub**: Recent repos or contribution graph placeholder
5. **Contact**: Email, social links (GitHub, LinkedIn, Twitter)

### Interactions
- Smooth scroll to sections
- Hover effects on project cards
- Animated skill icons

## Acceptance Criteria
- Page loads without errors
- All sections visible and properly styled
- Responsive on mobile and desktop
- Links are functional
